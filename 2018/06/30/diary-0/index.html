想起了几部动画片

Share

小学的时候，家里还没有电脑之前，我特别喜欢看电视。那时候电视上可以看到不少国外引进的动画片，相比于现在，我更喜欢那时的电视节目。如今已经不看电视了。

那时有一个法国出的动画片《Code Lyoko》，大陆翻译叫《虚幻勇士》，后来因为一些原因下架了。也许是受了这部动画片的影响，我对计算机和虚拟的美少女抱有好感。现在我依旧喜欢二次元老婆。

code lyoko 中文版的主题曲

噢 来吧 拼搏吧 为胜利的到达
噢 繁星 燃烧吧 将黑暗摧毁
勇敢吧 战胜它 守护这信念
让邪恶 远离这世界
我不孤单 有你守候
我更坚强 为你加油
我不后退 受你鼓舞
用爱点燃 希望之光

这歌还是不错的，很单纯，没有掺杂社会主义核心价值观，我只能怎么评价了。
（大家感受一下国内的校歌和国外的校歌有何不同，或者国内的校歌有何相同之处）

《百变小樱魔术卡》《光能勇士》《神龙斗士》《中华小当家》《马丁的早晨》《小魔女蒙娜》《猫和老鼠》《哆啦A梦》等等一批优秀的动画片。（《御兽王者》还行）

初中的时候，喜欢看《七色战记》《莫莫》《缇可》《蔷薇少女》《蓝兰岛漂流记》《名侦探柯南》。

《七色战记》是原创动力推出的，和《喜羊羊与灰太狼》是同一个公司。《七色战记》就一季，喜羊羊不知道是多少季了。 :-)

七色战记 主题曲

天大地大 树木和鲜花
在这宇宙我们同样是一家
每一种颜色 是一种精华
都在奉献他们特别的光华
才让这个星球 能美丽如画

不知道 一颗善心 一点耐心
能否把纷争都转化
只希望一个笑容握握手
大家把仇恨放下
团结走到一起 七色光芒敌人也害怕

不要动武 不要再轰炸
和平友爱 从来就是好方法
分开就弱小 合作就强大
刀山火海千辛万苦 笑一下
能掌握七色力量就掌握世界
能拥有七色力量就不怕

因为我们是一家

这个也不错，喜欢它的理由同上。

《莫莫》，《缇可》这两个动画片也不错，视美动画出的，这个公司挺厉害的。

影响深刻的要属《蔷薇少女》，在这就不说了。

高中接触的动画片多是日本的，同学介绍了一部不错的动画片，于是入坑了，它叫《CLANNAD》。
不多评价，很多人或许都听过。（喜欢二次元的人）

《白兔糖》《寒蝉鸣泣之时》《钢之炼金术师》《颠倒的帕特玛》《一周的朋友。》《sola》《小魔女学园》《文学少女》《异国迷宫的十字路口》《人形电脑天使心》《回转企鹅罐》《尸鬼》等等。
《人形电脑天使心》才是真的人工智能！这阶段看的都是比较经典的

还有现在的看的《神灵狩》《玲音》，这个两个不错，推荐一下。

还有其它的，以后想起来了再说。

最近在看《世界名作剧场》的动画片，其中的《悲惨世界 少女珂赛特》真心不错，准备慢慢看。
